# Agente de Contatos

## Responsabilidade
Gerenciar a base de contatos do usuário.

## Funções
- Buscar todos os contatos
- Buscar contatos específicos
- Criar novos contatos
- Atualizar informações de contatos existentes

## Integrações
- Google Contacts API

## Estratégia de Funcionamento
Executa operações CRUD sobre contatos a partir das instruções do Agente Pessoal.

## Casos de Uso
- Atualização automática de contatos
- Centralização de informações pessoais e profissionais
