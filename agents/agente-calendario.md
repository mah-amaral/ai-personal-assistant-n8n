# Agente de Calendário

## Responsabilidade
Gerenciar eventos e compromissos do usuário no calendário.

## Funções
- Buscar eventos existentes
- Criar novos eventos
- Criar eventos com participantes
- Atualizar eventos
- Apagar eventos

## Integrações
- Google Calendar API

## Estratégia de Funcionamento
Executa operações de calendário com base na intenção interpretada pelo Agente Pessoal.

## Casos de Uso
- Agendamento automático de reuniões
- Organização de compromissos
- Manutenção da agenda atualizada
